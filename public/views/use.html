Use it

To use NaView in your website, all you need to do is include the following libraries in your project:


<script src="https://cdnjs.cloudflare.com/ajax/libs/d3/5.16.0/d3.min.js"></script>
<script src="https://cdn.jsdelivr.net/gh/marceloqla/NaView@1.1/public/js/naview.js"></script>

To draw the image, you need to create a NaView object passing a JSON settings argument, as can be seen in the example below. The protein input must be a UniProtKb sequence in the text format (i.e. P14555).


fetch("https://www.uniprot.org/uniprot/P14555.txt")
    .then(response => response.text())
    .then(data => {
    var naview = new NaView({
    	"protein_input":data, 
        "container_id": "result",
        "svg_width": 800
    });
});

The style editor can be enabled with following commands:

<script src="https://cdn.jsdelivr.net/gh/marceloqla/NaView@1.1/public/js/editor.js"></script>

var style_obj = naview.getStyleObject("json");
var style_editor = new NaViewStyleEditor(naview, "style_editor", "style_editor_console");
